# AI-Based Predictive Maintenance System for a DC Motor

## Project Overview

This Project develops a low-cost predictive maintenance system for a DC motor. Sensors are used to collect information about the motor's operating condition, including vibration, temperature, electrical current, and rotational speed. The collected data will be analyzed using Python and machine-learning techniques to identify abnormal behavior and predict potential motor problems before a complete failure occurs.

## Project Objectives 

- Build and operate a small DC motor testing system.
- Measure motor vibration, temperature, current, and rotational speed.
- Collect sensor measurements under normal and abnormal operating conditions.
- Store and visualize the collected data.
- Train a machine-learning model to classify the motor's condition.
- Create a practical engineering portfolio project demonstrating hardware, programming, data analysis, and predictive maintenance.

## System Concept
The project follows this general process:

1. Sensors measure the condition of the DC motor.
2. A microcontroller reads the sensor measurements.
3. The measurements are transmitted to a computer.
4. Python stores and analyzes the data.
5. A machine-learning model identifies normal and abnormal motor behavior.
6. The system produces a maintenance warning when abnormal behavior is detected.
